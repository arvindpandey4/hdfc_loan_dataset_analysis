# HDFC Loan Dataset Analysis

**Task Completed:** 26/12/2025

## Overview
Statistical analysis of HDFC loan dataset covering descriptive statistics, probability, hypothesis testing, and Bayes theorem.

## Key Concepts

### Data Processing
- CSV to JSON conversion with data cleaning
- Duplicate removal and validation of non-negative financial values

### Statistical Analysis
1. **Descriptive Statistics**: Mean, median, mode, variance, standard deviation, coefficient of variation
2. **Probability Fundamentals**: Joint, conditional, and marginal probabilities
3. **Distributions**: Histogram, KDE plots, Shapiro-Wilk normality test
4. **Correlation & Covariance**: Pearson correlation, covariance matrix analysis
5. **Hypothesis Testing**: Independent t-test, Chi-square test of independence
6. **Confidence Intervals**: 95% CI for mean estimates
7. **Law of Large Numbers**: Sample mean convergence demonstration
8. **Discrete Probability**: Bernoulli and Binomial distribution modeling
9. **Bayes Theorem**: Conditional probability verification

## Files
- `main.ipynb`: Complete analysis (22 questions)
- `data_processor.py`: Data cleaning script
- `source/hdfc_loan_dataset_cleaned.json`: Processed dataset
- `source/json.config`: Configuration file
